# theclassics.school
A text-based self-learning classroom for classic literature.

## Available courses
- How to Read a Book
- On Writing Well
- Aristotle's *Nicomachean Ethics*

Lessons are defined as YAML files in `lessons/` and registered in `lessons.json` for the UI to load.

## Getting started
1. Open `index.html` locally or via GitHub Pages.
2. Sign in with your email link to access lessons.
3. Your lesson progress is saved to Firestore per user.
